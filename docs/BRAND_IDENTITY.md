# Brand Identity

Authoritative brand reference for Nova Systems Lab. Public copy, UI, and assets
must conform to this document. See also `PRODUCT_POSITIONING.md` (what we say we
are) and `design/DESIGN_TOKENS.md` (the implemented values).

## Identity in one line

Nova Systems Lab is an independent, founder-led software organization building
systems tools, developer utilities, platform integrations, and experimental
runtime technologies.

- **Name:** Nova Systems Lab (never abbreviate to "Nova" alone in formal copy)
- **Tagline:** Open systems. Better tools. Independent software.
- **GitHub organization:** `Nova-Systems-Lab`
- **Founder account:** `SamratB8`

## How the identity should feel

Modern 21st-century technology startup; a serious engineering organization;
ambitious but honest and early-stage; product-led; developer-focused;
systems-oriented; technically credible; open-source friendly; and structured so
it can become commercial later without a rebrand.

## Brand personality

**Is:** precise · ambitious · transparent · independent · technical ·
thoughtful · experimental · disciplined · modern · approachable

**Is not:** arrogant · exaggerated · overly corporate · unserious · crypto-style ·
gamer-focused at the organization level · generic AI-generated startup voice

The brand earns credibility by showing working software and honest status, not
by claiming scale it does not have.

## Tone of voice

Write clearly, directly, and confidently. Be technically literate but
understandable to a non-specialist. Be honest about maturity. Prefer concrete
statements over adjectives.

**Banned phrases** (non-exhaustive): "revolutionizing technology", "changing the
world", "industry-leading", "world-class", "cutting-edge solutions", "disruptive
innovation", "trusted by thousands", and "enterprise-grade" unless demonstrably
true.

| Instead of                           | Write                                                    |
| ------------------------------------ | -------------------------------------------------------- |
| "Revolutionary developer platform"   | "Developer tools for Windows and Linux workflows"        |
| "Trusted by thousands of developers" | "Open source on GitHub"                                  |
| "Enterprise-grade runtime"           | "Runtime currently in early development"                 |
| "Cutting-edge WSL solution"          | "WSL Studio — a workspace for managing WSL environments" |

Rules: never claim legal incorporation; never invent partners, customers, team
members, testimonials, metrics, or funding; never label maturity without
evidence.

## Visual direction

Dark-first, high-contrast, restrained. Graphite, deep navy and near-black
surfaces; white and cool grey text; **electric blue as the single organization
accent**. Cyan, violet, and green appear only as restrained status accents.

The organization palette, typography, spacing, radii, shadows, focus rings,
motion, breakpoints, content widths, grid, and icon style are specified with
concrete values — and measured contrast — in `design/DESIGN_TOKENS.md`.

## Product accents

Product accents are subordinate to the organization identity. A product accent
may tint a product page's highlights; it must never replace the electric-blue
organization accent in global chrome (header, footer, primary actions).

| Product                       | Accent direction                                          | Status                                           |
| ----------------------------- | --------------------------------------------------------- | ------------------------------------------------ |
| WSL Studio                    | Windows/terminal blue                                     | Not yet implemented                              |
| WinDroid Runtime              | Restrained Windows blue + Android-inspired green          | Not yet implemented                              |
| Future game studio / RigForge | Separate identity, kept fully apart from Nova Systems Lab | Not operational; must not be presented as active |

Product accent tokens are intentionally **not** implemented yet — they will be
added when real product pages exist.

## Prohibited design practices

Never ship: excessive glassmorphism · excessive gradients · fake 3D globes ·
fake customer logos · fake testimonials · fake usage metrics · fake funding
badges · fake employee photos · stock-office imagery · autoplay video · flashing
animation · excessive parallax · text hidden for visual effect · low-contrast
grey text · emoji as interface icons · decorative terminal windows containing
meaningless commands · dead `href="#"` · external CDN-loaded fonts or scripts ·
inline JavaScript · inline CSS.

## Assets

No approved logo, wordmark, or icon set exists yet. Do not invent, generate, or
substitute brand assets. The site currently ships the framework default
favicon — replacing it requires an approved brand asset (tracked in
`MISSION_STATUS.md`).
